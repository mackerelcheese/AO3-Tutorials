Mm whatcha say?: Changing Text On Hover for Multilingual Fics (A Download Friendly Workskin)

Summary

Do you want to have a nifty way to have multiple languages in a fic:
  • Without making your readers scroll to the AN for a translation?
  • Without the text and translation sitting side by side?
  • And still make it look pretty when people download it into an EPUB or PDF format?
Here is a solution for you. Come on babygirls, let's roll.

The Code

okay so this is just the code I used in the scene above:

The CSS (aka the thing to throw into your Workskin) 
.l1 {
  font-style:italic;
}

.l2 {
  display:none;
  font-style:italic;
}

.hide {
  display:none;
}

.translate:hover .l1 {
  display:none;
}

.translate:hover .l2 {
  display:inline;
}

Okay so now let's see it in action

The HTML (aka the thing to throw into your work body) 
<p>"Regulus," Sirius' hand tightens around Remus' upper arm. "I didn't think you were coming today."</p>
<p class="translate">"<span class="l1">怎么了?</span><span class="hide"> (</span><span class="l2"><em>Why?</em></span><span class="hide">)</span>" he asks, tossing his sunglasses to the side. "<span class="l1">你不想见到我吗?</span><span class="hide"> (</span><span class="l2"><em>Don't want to see me?</em></span><span class="hide">)</span>"</p>
<p class="translate">Sirius clenches his jaw, "<span class="l1">你那是什么意思?</span><span class="hide"> (</span><span class="l2">What do you mean by that?</span><span class="hide">)</span>"</p>

Code Breakdown

You might be thinking woah Mack that's a lot. Yes, I'm aware but I'll hold your hand the whole time okay babygirls? You're doing so well. I'm so proud of you. You got this, I believe in you. (feel free to drop a comment on this work if you're stuck at any point)

Let's take this part by part:

  • translate: refers to the sentence that has the multilingual text
  • l1: refers to the language you want displayed in your fic as is.
  • l2: refers to the language you want displayed when the reader hovers/clicks on the text
  • hide: refers to the sections that will only be visible when the fic has been downloaded (another La_Temperanza trick)
  • em: a HTML way to make text italic

Sentence Structure

Making the hovers work:

First thing you need to note is the whole sentence. Instead of putting your sentence in regular <p></p> you have to use this <p class="translate"></p>. This acts as the hoverable element.

Next you have to add language sections, now it'll look like this:

<p class="translate"><span class="l1">this is your language one</span><span class="l2">second language looks like this</span></p>

It'll now look something like this with creator style on:

this is your language onesecond language looks like this

Cool, now your text is hoverable and changable, good job you.

Making it look pretty when you download it/creator style off:

With creator style off (aka when you download it), it will look like this:

this is language onesecond language looks like this

uhhhh okay less good.

Okayyy so let's fix that. But here comes the confusing part. Shhhh its okay, I gotchu. I'm going to hold your hand through this. We will get it together.

Here is the confusing part: the <span class="hide"></span>

The rationale behind this part is to hide the () that will appear around your L2 once your work gets downloaded as a PDF or EPUB.

I want the text to look like so once Creator Style is turned off or when the fic has been downloaded:

this is language one (second language looks like this).

So with that in mind, we have to do the scary bit now: *holds your hand*

  • To hide the brackets, you have to use <span class="hide"></span> to hide the brackets
  • To have the language two be in italics even with creator style/downloaded you have to wrap the affected text in <em></em>

Super duper important things to note:

  • So one thing I've learned while coding is that SPACES AND ORDER MATTER especially in <span></span>s. Trust me, I've cried over this so you know its legit. This means you have to be wary about the spaces between your brackets, text, punctuation, quotation marks, etc.
  • You can put whatever text you want in the same paragraph, just make sure its outside of the affected <span></span>s.

Okay so let's put it together:

Code breakdown:

 
<p class="translate">He sneezed before talking. "<span class="l1">this is language one</span><span class="hide"> (</span><span class="l2"><em>second language looks like this</em></span><span class="hide"> )</span>", said your character.</p>

And this is how it will look like with Creator Style on:

He sneezed before talking. "this is language one (second language looks like this )", said your character.

Without Creator Style on or downloaded, it should look like this:

He sneezed before talking. "this is language one (second language looks like this)", said your character.
